# Foosha - Food Surplus & Donation Logistics Platform

Foosha is a full-stack web application connecting food donors, volunteer couriers, and charity relief centers to eliminate food waste and feed communities.

## Repository Structure

- **`foosha/`**: The main Next.js web application directory.

### Quick Start

Navigate to the application directory and start the dev server:

```bash
cd foosha
npm install
npm run dev
```
