# Foosha - Food Surplus & Donation Logistics Platform

Foosha is a modern, full-stack web platform built to bridge the gap between surplus food and hunger. By coordinating and matching local food donors (restaurants, supermarkets, caterers) with volunteer riders and community relief centers, Foosha turns potential food waste into immediate nutritional support.

## Features

- **Surplus Food Logging**: Donors can list food details, ingredients, expiry times, and portion sizes with multi-step validation.
- **Relief Demand Queues**: Charity centers and relief workers can post dietary needs and headcount targets.
- **Auto-Matching Alliance**: Backend logic pairs surplus donations with nearest recipient queues and dispatches volunteer couriers.
- **Secure Authentication**: Integrated with Supabase Auth and Edge Proxy protection.
- **Responsive & Dynamic UI**: Built with Next.js App Router, React 19, and Tailwind CSS.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to explore Foosha.
